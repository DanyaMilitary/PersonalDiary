## Описание

**«Система управления успеваемостью»** — это десктопное приложение для автоматизации учёта успеваемости и посещаемости в школе, разработанное в рамках учебной практики в колледже. Приложение позволяет директору управлять учителями и учениками, учителям — выставлять оценки и вести посещаемость, а ученикам — просматривать свои оценки и статистику в удобной визуальной форме.

## Возможности

- **Три роли пользователей:** директор, учитель, ученик
- **Управление пользователями:** добавление и удаление учителей и учеников
- **Выставление оценок** (1–5) с комментариями
- **Учёт посещаемости** (присутствовал, отсутствовал, опоздал)
- **Просмотр успеваемости** в виде прогресс-баров по предметам
- **Статистика** среднего балла и посещаемости
- **Отладочные инструменты** (лог, системный монитор) — горячая клавиша `Ctrl+Shift+D`
- **Локальная база данных SQLite** — все данные сохраняются на компьютере

## Установка

1. Убедитесь, что на компьютере установлена **Java 11** или выше (рекомендуется IntelliJ IDEA Community Edition).
2. Скачайте все `.java` файлы приложения и драйвер базы данных `sqlite-jdbc.jar`.
3. Поместите `sqlite-jdbc.jar` в папку с проектом или добавьте его в classpath.
4. Скомпилируйте и запустите файл `Main.java` (или `LoginUI.java`).
5. При первом запуске автоматически создастся файл базы данных `diary.db` и начальные данные.

## Использование

**Система управления успеваемостью** — это многофункциональное приложение для школ, построенное на ролевой модели доступа. Оно предоставляет удобные инструменты для ведения журнала успеваемости и контроля посещаемости.

### Начальный вход
- **Директор:** логин `director`, пароль `director123`
- Далее учителей и учеников добавляет директор / учителя соответственно

### Роли пользователей

| Роль | Основные возможности |
|------|----------------------|
| Директор | Добавление/удаление учителей, просмотр всех учеников |
| Учитель | Добавление учеников, выставление оценок, отметка посещаемости, история посещаемости |
| Ученик | Просмотр всех оценок, фильтрация по предметам, график успеваемости, посещаемость |
